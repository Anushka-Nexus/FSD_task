TRANSITION PROPERTIES OF CSS

 2-D TRANSFORMATION

 property:-
   transform : function;
     
     function
    1. scale :- it is used to grow or shrink the element
      scale();
      scaleX();//WIDTH
      scaleY();//height
      scale(x,y);
    
    2.translate(); :- it is used for moving the element 
     translate(trlb);
     here 
     translate(+ve value,+ve value); ye move krege towards right and bottom ki taraf

      translate(-ve value,-ve value); ye move krege towards left and top ki taraf

      translateX(value); //horizontal
      translatey(value);  //vertical
      

      ex:-
      .child
      {
        width:50px;
        height:50px;
        top:50%;
        left:50%;
        transform:translate(-50% , -50%);

      }

    3.rotate(); :- this function is used for rotating the element.

     here there are three functions :
      (i)rotate(); :- ye move krega along with its phase //PHASE MOVE
        format :-
         transform: rotate(0deg); positive value di to clockwise move krega
                                  negative value di to anticlockwise move krega

      (ii)rotateX(); :- ye move krega as a fallback //X AXIS 
         format:-
            transform: rotatecX(0deg); positive value di to aage move krega
                                  negative value di to piche move krega

     (iii)rotateY(); :- ye move krega as a flip //Y AXIS
         format:-
            transform: rotatecY(0deg); positive value di to left move krega
                                  negative value di to right move krega

    4.skew(); :- used to strecth the element
